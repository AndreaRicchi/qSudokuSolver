# qSudokuSolver

qSudokuSolver is a QML application that solves sudoku with backtracking algorithm.

## Build

Download the repo and build it.
```bash
git clone https://github.com/AndreaRicchi/qSudokuSolver.git
mkdir build && cd build
make -j2
./qSudokuSolver
```
